Pour exécuter notre application veuillez vous positionner dans le dossier BRAHIMI_ADDOUCHE ( apres la decompression du fichier .zip) et saisir la commande make.
Pour supprimer les fichiers avec extensions ".o" veuillez saisir make clean.
Notre projet est composé de fichiers main.c et main.h : le fichier .h contiendra les signatures des fonctions.
le fichier main.c contiendra l'ensemble des fonctions demandées dans le dm, c'est-à-dire Square and Multiply, Test de Fermat et Miller Rabbin, ainsi que l'interface de bord de notre application, utiliser pour l'interaction de l'utilisateur avec nos algorithmes et lui afficher les divers résultats souhaités.
